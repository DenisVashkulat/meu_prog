Завдання №1: Розробка макету власного сайту

Макет мого сайту використовує тип блокової верстки засобами CSS та Flexbox. Я використав флекс-контейнери для організації структури сторінки, що дозволяє ефективно розподіляти і розміщувати елементи на веб-сторінці. Flexbox надає зручний і потужний спосіб для роботи з макетами, що адаптивні до різних розмірів екранів.

Завдання №2: Створення окремих веб-сторінок

Таблична верстка:

Фіксована таблична верстка: Використовуючи таблицю, я здійснив фіксовану табличну верстку. Цей підхід дозволяє точно контролювати розмір і положення елементів, але не є адаптивним до різних розмірів екрану. Гумова таблична верстка: Таблична верстка з використанням відсоткових значень для розмірів елементів. Цей метод забезпечує певну гнучкість, але не таку, як у випадку блокової верстки.

Блокова верстка (плаваючі елементи):

Фіксована блокова верстка: Використання фіксованих ширин елементів з float або grid дозволяє точно розташовувати елементи, але не є адаптивним до зміни розмірів екрану. Гумова блокова верстка: Використання відсоткових значень або flexbox для розміщення елементів, що забезпечує певну гнучкість та адаптивність до різних розмірів екрану.

Завдання №3: Використання CSS та Flexbox

У виконанні цього завдання я використав Flexbox для створення адаптивної сторінки. Flexbox дозволяє легко контролювати розташування елементів, розподіляти простір між ними і змінювати їх порядок відносно розміру екрану. Цей підхід спрощує створення адаптивних дизайнів без необхідності використання складних розрахунків для положення елементів.

Висновки

Таблична верстка має свої переваги у точному контролі над розміщенням елементів, але не є оптимальним варіантом для адаптивних дизайнів. Блокова верстка (плаваючі елементи) є більш сучасним підходом, який дозволяє створювати адаптивні сторінки, але потребує використання додаткових технік для точного розташування елементів. Flexbox є потужним і зручним інструментом для створення адаптивних макетів. Він спрощує розташування елементів і забезпечує гнучкість у керуванні простором на сторінці. Таким чином, під час виконання лабораторної роботи я отримав практичні навички засобами CSS та Flexbox, що дозволяють створювати сучасні та адаптивні веб-сторінки.
