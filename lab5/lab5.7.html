Завдання:

Динамічне меню

Сформувати багаторівневе меню (горизонтальне та вертикальне), пункти меню взяти із статичного масиву. При натисканні на визначений пункт меню має завантажуватися відповідна HTML-сторінка. Меню має бути присутнім на всіх сторінках.

Виконання коду:

Код:

        <!DOCTYPE html>
        <html lang="en">
        <head>
            <meta charset="UTF-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Меню</title>
            <style>
                /* Стилі для горизонтального меню */
                .horizontal-menu {
                    width: 350px;
                    height: 30px;
                    display: flex;
                    list-style-type: none;
                    background-color: #f4f4f4;
                    padding: 10px;
                    margin: 0;
                }
                .horizontal-menu li {
                    margin-right: 20px;
                }
                .horizontal-menu li:last-child {
                    margin-right: 0;
                }
        
                /* Стилі для вертикального меню */
                .vertical-menu {
                    width: 100px;
                    height: 90px;
                    display: flex;
                    flex-direction: column;
                    list-style-type: none;
                    background-color: #f4f4f4;
                    padding: 10px;
                    margin: 0;
                }
                .vertical-menu li {
                    margin-bottom: 20px;
                }
                .vertical-menu li:last-child {
                    margin-bottom: 0;
                }
        
                /* Загальні стилі для меню */
                .menu-link {
                    text-decoration: none;
                    color: #333;
                    transition: color 0.3s;
                }
                .menu-link:hover {
                    color: rgb(0, 68, 255);
                }
                header{
                    display: flex;
                    justify-content: center;
                }
            </style>
        </head>
        <body>
            <header><h1>Це перша сторінка</h1></header>
        
            
            <ul class="horizontal-menu">
                <li><a class="menu-link" href="home.html">Пункт меню 1</a></li>
                <li><a class="menu-link" href="page2.html">Пункт меню 2</a></li>
                <li><a class="menu-link" href="page3.html">Пункт меню 3</a></li>
            </ul>
        
            
            <ul class="vertical-menu">
                <li><a class="menu-link" href="home.html">Пункт меню 1</a></li>
                <li><a class="menu-link" href="page2.html">Пункт меню 2</a></li>
                <li><a class="menu-link" href="page3.html">Пункт меню 3</a></li>
            </ul>
        
            <script>
                // Отримання всіх посилань меню
                var menuLinks = document.querySelectorAll('.menu-link');
        
                // Додавання обробника подій для кожного посилання
                menuLinks.forEach(function(link) {
                    link.addEventListener('click', function(event) {
                        // Відміна стандартної дії посилання
                        event.preventDefault();
        
                        // Отримання URL посилання
                        var url = this.getAttribute('href');
        
                        // Перехід на відповідну сторінку
                        window.location.href = url;
                    });
                });
            </script>
        </body>
        </html>
