JavaScript

Функціональне застосування JavaScript охоплює багато аспектів, включаючи обробку подій, маніпуляції з DOM, роботу з масивами та об'єктами, анімацію, взаємодію з сервером і багато іншого. Ось деякі основні способи функціонального використання JavaScript:

1. Обробка подій

JavaScript використовує обробники подій для реагування на дії користувача або системні події. Це може бути клік на кнопку, введення тексту в поле, наведення курсору миші на елемент тощо.

        document.getElementById("myButton").addEventListener("click", function() {
        alert("Кнопку натиснуто!");
      });

2. Робота з DOM

JavaScript використовується для зміни структури та змісту сторінки в реальному часі, що дозволяє додавати, видаляти або змінювати елементи HTML.

        // Додати новий елемент до DOM
        let newElement = document.createElement("div");
        newElement.textContent = "Новий елемент";
        document.getElementById("container").appendChild(newElement);
        
        // Змінити вміст елементу
        document.getElementById("myParagraph").textContent = "Новий текст";
        
        // Видалити елемент
        let elementToRemove = document.getElementById("toBeRemoved");
        elementToRemove.parentNode.removeChild(elementToRemove);

3. Маніпуляції з масивами та об'єктами

JavaScript використовується для роботи з масивами та об'єктами, дозволяючи виконувати операції з даними.

        // Масив
        let myArray = [1, 2, 3, 4, 5];
        let doubledArray = myArray.map(function(num) {
          return num * 2;
        });
        console.log(doubledArray); // [2, 4, 6, 8, 10]
        
        // Об'єкт
        let myObject = { name: "John", age: 30, city: "New York" };
        console.log(myObject.name); // John
        myObject.age = 31;
        console.log(myObject.age); // 31

4. Взаємодія з сервером

JavaScript дозволяє виконувати асинхронні запити на сервер і отримувати дані, що дозволяє оновлювати сторінку без перезавантаження.

        fetch('https://api.example.com/data')
        .then(response => response.json())
        .then(data => {
          console.log(data);
          // Обробка отриманих даних
        })
        .catch(error => {
          console.error('Помилка:', error);
        });

5. Анімація

JavaScript використовується для створення анімаційних ефектів на сторінці.

        let element = document.getElementById("myElement");

        function moveElement() {
          let position = 0;
          let id = setInterval(frame, 10);
        
          function frame() {
            if (position == 350) {
              clearInterval(id);
            } else {
              position++;
              element.style.top = position + "px";
              element.style.left = position + "px";
            }
          }
        }
        
        element.addEventListener("click", moveElement);
