ПОСТАНОВКА ЗАДАЧІ Лабораторної №6-7

Способи застосування об'єктів.

Завдання

Висновок

Застосування об'єктів.

Об'єкти в JavaScript є одними з основних структурних елементів мови, які дозволяють організувати та зберігати дані у вигляді пар ключ-значення. Вони використовуються для різних цілей і мають ряд способів застосування:

1. Зберігання та організація даних: Об'єкти дозволяють структурувати дані у вигляді властивостей (або полів), кожна з яких має унікальний ключ.

Наприклад:

        let person = {
            name: 'John',
            age: 30,
            address: {
                city: 'New York',
                country: 'USA'
            }
        };

2. Представлення реальних об'єктів: Об'єкти можуть бути використані для моделювання реальних об'єктів або сутностей, що мають характеристики та поведінку.

Наприклад:

        // Моделювання автомобіля
        let car = {
            brand: 'Toyota',
            model: 'Corolla',
            year: 2020,
            start: function() {
                console.log('Двигун запущено');
            },
            stop: function() {
                console.log('Двигун зупинено');
            }
        };
        
        // Виклик методів об'єкта
        car.start();
        car.stop();

3. Моделювання комплексних даних: Об'єкти дозволяють створювати складні структури даних, які можуть включати інші об'єкти, масиви, функції та інші дані. Це забезпечує гнучкість та ефективне управління даними.

Наприклад:

        let student = {
            name: 'Alice',
            age: 25,
            grades: [85, 90, 95],
            address: {
                city: 'London',
                country: 'UK'
            },
            getAverageGrade: function() {
                let sum = this.grades.reduce((total, grade) => total + grade, 0);
                return sum / this.grades.length;
            }
        };
        
        console.log(student.getAverageGrade()); // Виведе середній бал студента
