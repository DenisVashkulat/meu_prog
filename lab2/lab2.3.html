Файл style.css:

  1. body - Встановлює стилі для тіла HTML-документу.
  2. header - Стилізує заголовок сторінки.
  3. main - Стилізує основний контент сторінки.
  4. footer - Стилізує нижній колонтитул сторінки.
  5. img - Стилізує всі зображення.
  6. table - Стилізує всі таблиці.
  7. th - Стилізує заголовки таблиць (th).
  8. td - Стилізує клітинки таблиць (td).
  9. form - Стилізує всі форми.
 10. label - Стилізує всі підписи (label) у формах.
 11. input - Стилізує всі поля введення (input).
 12. a.perehid - Це селектор класу, але він застосований до тегу a (посилання).
 13. img.icons - Це селектор класу, але він застосований до тегу img (зображення).

Файл stylemain.css:

  1. body - Встановлює стилі для тіла HTML-документу.
  2. header - Стилізує заголовок сторінки.
  3. main - Стилізує основний контент сторінки.
  4. footer - Стилізує нижній колонтитул сторінки.
  5. img.icons - Це селектор класу, але він застосований до тегу img (зображення).
  6. ol та li - Стилізують впорядковані списки (ol) та їх елементи (li).
  7. h2 - Стилізує заголовки другого рівня (h2).
  8. fieldset - Стилізує тег fieldset.
