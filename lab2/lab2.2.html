CSS селектори є ключовими частинами стилізації веб-сторінки, оскільки вони дозволяють вибирати елементи HTML, до яких потрібно застосувати стилі. Існує кілька різних типів селекторів у CSS, кожен з яких дозволяє по-різному вибирати елементи:

    1. Селектори за тегом (Type Selectors):
        Вибирають елементи HTML за назвою тегу.
        Наприклад: p для вибору всіх тегів <p>.
    2. Селектори за класом (Class Selectors):
        Вибирають елементи за їхнім класом.
        Для вибору використовується крапка (.) перед назвою класу.
        Наприклад: .my-class для вибору всіх елементів з класом my-class.
    3. Селектори за ідентифікатором (ID Selectors):
        Вибирають елементи за їхнім унікальним ідентифікатором.
        Для вибору використовується решітка (#) перед назвою ідентифікатора.
        Наприклад: #my-id для вибору елемента з ID my-id.
    4. Селектори за атрибутом (Attribute Selectors):
        Вибирають елементи за їхніми атрибутами.
        Наприклад: [type="text"] вибирає всі елементи з атрибутом type="text".
    5. Селектори нащадків (Descendant Selectors):
        Вибирають елементи, які є нащадками іншого елемента.
        Наприклад: div p вибирає всі теги <p> всередині тегу <div>.
    6. Селектори дочірніх елементів (Child Selectors):
        Вибирають елементи, які є безпосередніми дочірніми елементами іншого елемента.
        Наприклад: div > p вибирає всі теги <p>, які є безпосередніми дочірніми елементами тегу <div>.
    7. Селектори сусідніх елементів (Adjacent Sibling Selectors):
        Вибирають елемент, що йде одразу після іншого елемента на одному рівні.
        Наприклад: h1 + p вибирає тег <p>, що йде одразу після тегу <h1>.
    8. Селектори наступних елементів (General Sibling Selectors):
        Вибирають усі наступні елементи після певного елемента на одному рівні.
        Наприклад: h1 ~ p вибирає всі теги <p>, які йдуть після тегу <h1> на одному рівні.
    9. Псевдокласи (Pseudo-classes):
        Використовуються для вибору елементів на основі їхнього стану, позиції в DOM чи інших критеріїв.
        Наприклад: a:hover для стилізації посилання, коли користувач наводить на нього курсор.
   10. Псевдоелементи (Pseudo-elements):
        Використовуються для вибору та стилізації певних частин елементів, наприклад, першої літери чи першого рядка.
        Наприклад: p::first-line для вибору першого рядка тегу <p>.</pre>
