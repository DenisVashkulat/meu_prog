Висновки до ЛР№2

Селектори ідентифікаторів в CSS використовуються для стилізації елементів HTML, які мають певний унікальний атрибут id. Унікальність цього атрибуту дозволяє легко знаходити конкретний елемент на сторінці та застосовувати до нього стилі. Селектори ідентифікаторів в CSS починаються з символу # перед іменем ідентифікатора. Ось приклади селекторів ідентифікаторів та їх використання:

  1. Селектор ідентифікатора для заголовку:

  #header {
    background-color: #f8f8f8;
    color: #333;
    padding: 10px;
  }

В цьому прикладі стилізується елемент HTML з ідентифікатором header. До нього застосовуються стилі фону, кольору тексту та відступів.

  2. Селектор ідентифікатора для головного розділу:

  #main {
    padding: 20px;
    font-size: 16px;
  }

В цьому прикладі стилізується елемент HTML з ідентифікатором main. До нього застосовуються стилі відступу та розміру шрифту.

  3. Селектор ідентифікатора для форми:

  #signup-form {
    width: 300px;
    padding: 10px;
    border: 1px solid #ccc;
  }
  
 Цей приклад стилізує форму (<form>) з ідентифікатором signup-form. Стилі включають ширину, відступи та рамку.
