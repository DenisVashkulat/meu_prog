Висновки до ЛР№2

Крім селекторів тегу, класу та ідентифікатора, у CSS є багато інших видів селекторів, які ви можете використовувати для більш точного або специфічного стилізування елементів HTML. Ось деякі з них:


  1. Селектори атрибутів:
     Селектори, які вибирають елементи на основі їхніх атрибутів.
     Приклади:


      /* Елементи з атрибутом type="text" */
input[type="text"] {
    border: 1px solid #ccc;
}

/* Елементи з атрибутом title, який містить "example" */
a[title*="example"] {
    color: red;
}



  2. Псевдокласи:
     Псевдокласи використовуються для вибору елементів у певному стані, наприклад, коли користувач наводить мишку або активує елемент.
     Приклади:


 /* Посилання, коли на них наводиться мишка */
a:hover {
    color: blue;
}

/* Елементи форми, які мають фокус */
input:focus {
    outline: none;
}


  3. Псевдоелементи:
     Псевдоелементи дозволяють стилізувати частини елементів.
     Приклади:


/* Додає контент перед елементом */
p::before {
    content: "• ";
}

/* Зміна стилю першої літери елемента */
p::first-letter {
    font-size: 2em;
    color: blue;
}



  4. Групування селекторів:
     Ви можете групувати кілька селекторів для застосування однакових стилів до різних елементів.
     Приклад:


/* Застосовує один і той же стиль до h1, h2, та h3 */
h1, h2, h3 {
    color: green;
}



  5. Комбіновані селектори:
     Комбіновані селектори дозволяють вибирати елементи, які знаходяться в певному відношенні до інших елементів.
     Приклади:



/* Дочірні елементи ul, що є li */
ul > li {
    color: red;
}

/* Елементи li, які слідують за іншими li */
li + li {
    margin-top: 10px;
}



  6. Універсальні селектори:
     Селектор * вибирає всі елементи на сторінці.
     Приклад:


/* Застосовує рамку до всіх елементів */
* {
    border: 1px solid #000;
}
