Висновки до ЛР№2

1. Вбудований стиль (Inline CSS): Ви можете застосувати стилі безпосередньо до елементів HTML, використовуючи атрибут style.

Наприклад: 

 <div style="color: red;">Текст</div>

Це підходить для стилізації окремих елементів, але не є оптимальним способом для стилізації всієї сторінки.

2. Вбудований блок стилів (Internal CSS): Ви можете додати стилі безпосередньо в документ HTML у блоці <style>, розміщеному в розділі <head>. 

Наприклад:
  
  <head>
  <style>
        body {
            background-color: #f0f0f0;
        }
        h1 {
            color: blue;
        }
  </style>
  </head>

Цей підхід підходить для невеликих проектів або для стилізації окремої сторінки.

3. Зовнішній файл стилів (External CSS):

Стилі зберігаються в окремому файлі з розширенням .css і підключаються до HTML-сторінки за допомогою тегу <link>. 

Наприклад:
  
    <head>
    <link rel="stylesheet" href="styles.css">
    </head>

Це найбільш поширений і кращий спосіб для підключення стилів, оскільки дозволяє розділити структуру (HTML) і стилі (CSS) та спростити обслуговування та оновлення стилів.

4. Імпорт стилів (Importing Styles):

Ви можете імпортувати зовнішній файл стилів в інший файл стилів за допомогою директиви @import.

Наприклад: 
      
        @import url("styles.css");

Це дозволяє організувати стилі в кілька файлів, що може спростити їх обслуговування.
