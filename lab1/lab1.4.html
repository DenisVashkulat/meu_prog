Теги форм


        <form>: Цей тег визначає форму на веб-сторінці. Всі елементи форми, такі як тексти, кнопки, радіокнопки, і т. д., розміщуються всередині цього тегу.

        <input>: Цей тег використовується для створення різноманітних елементів введення, таких як текстові поля, радіокнопки, прапорці, кнопки тощо.
        
        <textarea>: Цей тег створює багатострокове текстове поле, в якому користувач може вводити більше одного рядка тексту.
        
        <select>: Цей тег використовується для створення списку випадаючих. Він містить один або декілька тегів <option>, які представляють елементи списку.

Атрибути форм


        action: Цей атрибут вказує URL-адресу або шлях на сервер, до якого буде відправлено дані з форми після натискання кнопки відправлення.

        method: Цей атрибут визначає метод HTTP, який буде використовуватися для відправлення даних форми на сервер.
        Найпоширеніші значення цього атрибуту - "get" та "post".

        name: Цей атрибут вказує ім'я форми, яке можна використовувати для звернення до неї зі скриптів або CSS.

        value: Цей атрибут вказує значення, яке буде відправлено на сервер, коли користувач вибере або введе дані в елементі форми.
